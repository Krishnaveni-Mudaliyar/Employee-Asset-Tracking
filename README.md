## Current baseline

The baseline includes the asset master, employee assignment and return flow, maintenance, audit log, reporting queries, report layout, role center, permission sets, API pages, approval-request foundation, workflow policy seam, and warranty-monitoring job queue codeunit.

The feature-oriented target structure and planned AppSource-quality object catalog are defined in `docs/FILE_ARCHITECTURE.md`. Planned items remain documented until implemented and tested; they are not represented by compile-breaking empty AL files.
